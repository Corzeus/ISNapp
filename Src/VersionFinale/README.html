Read-me

Voiçi le mode d’emploi pour l’utilisation de notre application.

Prérequis

Pour lancer l’application il faut avoir Python 3 (la versino compte peu je pense) et les librairies suivantes:

  • "lxml" et "xml" : Elles sont facultatives si vous n’utilisez pas la fonctionnalité génération de plan qui se trouve dans “GenPlan.py”
  • "cgi" : Permet le décriptage des paquets envoyés par protocole http
  • "urllib" : Permet de décrypter facilement les urls

Ensuite il vous suffit dans le terminal de commandes de lancer le fichier Server.py puis d’aller dans votre navigateur web et tapper l’url suivante : “localhost:8000/”
Si jamais: J’ai aussi fait une compilation du fichier “Serveur.py”, “Serveur.exe” mais je n’ai aucune idée de si ça marche sous linux ou non, j’ai seulement pu essayer sous windows. C’est tout de fois, une meilleure manière de lancer le programme sans embûches sous windows

Génération de plan

Si vous voulez tester la fonctionnalité voilà comment il faut utiliser le programme:

  1. Faire un plan sous forme Svg: Le programme associe les rectangles ayant une id commençant par “Chambre” (pour modifier l’id dans inkscape il faut aller dans : Objets-> Propriétés de l'objet, puis un pannel s’affiche ensuite à droite montrant les caractéristiques de l’objet séletionné , modifiez ensuite la propriété id)comme une chambre, ainsi lorsque vous allez créer le plan il faudra mettre autant de rectangle que de chambres dans le plan (vous pouvez rajouter du texte par-dessus comme dans l’exemple de notre page, ça n’affecte pas). Pour créer des fichiers svg je recommande l’utilisation de Inkscape (gratuit, open-source et c’est ce que nous avons utilisé durant notre projet). Important : lorsque vous allez créer, redimensionnez bien la taille de votre dessin à votre contenu une fois le dessin terminé (fichier -> propriétés du doc -> "ajuster la page au dessin ou à la selection", pour inkscape) sinon la mise en page risque de ne pas fonctionner.
  2. Lancer le programme : Il suffit d’exécuter le programme “GenPlan.py” avec python, il va ensuite falloir rentrer un chemin vers votre plan puis le nom du plan de sortie (sans l’extension)
  3. Consultation de la page: vous pouvez consulter le plan en lançant le server et en tappant "localhost:8000/Plans/NomDeVotrePlan.html"
